Jocul in versiunea release se deschide cu fisierul "GAME.exe".

Scopul jocului este sa se atinga un scor cat mai mare distrugand inamicii.

La fiecare 50 de puncte se primeste o noua viata daca nu avem deja numarul maxim de vieti.

Jocul se joaca de pe sageti pentru deplasare si space pentru a folosi laserul.

Acest proiect a fost facut pentru atestatul de competente digitale de pe a 12-a, asadar se observa nerespectarea unor tehnici de programare care vizeaza scalabilitatea si readability-ul proiectului, folosite in OOP.
